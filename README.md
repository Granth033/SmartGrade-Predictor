# SmartGrade-Predictor
"SmartGrade-Predictor is a predictive machine learning model that automates the student evaluation process. It evaluates 13 key academic and demographic parameters to predict final grades (A to F), providing users with real-time performance insights and a detailed student profile summary through an interactive Streamlit dashboard."

# 🎓 Pro Student Grade Predictor

A robust Machine Learning application built using Python, scikit-learn, and Streamlit. This app analyzes student parameters (like weekly study time, absences, and current GPA) using a trained classification model to predict academic performance with a responsive and modern UI.

## 💻 Tech Stack
* **UI Framework:** Streamlit
* **Machine Learning:** Scikit-Learn
* **Data Processing:** Pandas, NumPy
* **Language:** Python

## 🛠️ How to Run Locally

1. Clone the repository:
   git clone https://github.com/tera-username/SmartGrade-Predictor.git
   
2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   streamlit run app.py
